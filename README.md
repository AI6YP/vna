# vna
miniVNA Tiny VNA setup

### Setup OpenSuse

```sh
sudo usermod -a -G tty yourname
sudo usermod -a -G dialout yourname

cd programs
wget http://download.dl2sba.com/vnaj/3.2.7/vnaJ.3.2.7.jar
java -Dsun.java2d.uiScale=2 -jar vnaJ.3.2.7.jar
```
